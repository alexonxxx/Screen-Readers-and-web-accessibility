    Introducción

    Los screen Readers son un software que permite utilizar el sistema operativo principalmente para que se explique o lea lo que hay por pantalla, con el objetivo de ayudar a personas con problemas de visión.

    En el mercado podemos encontrar bastantes screen readers algunos gratuitos y otros de pago. Tanto los gratuitos como los de pago suelen leer el texto que hay en la pantalla donde se sitúa el ratón o de donde clicas, pero los screen readers de pago además añaden nuevas funcionalidades como diferentes voces disponibles, leer lo que se escribe para un dispositivo Braille o señalar de una manera más potente para seleccionar donde estás ubicado de la pantalla.

    Screen Readers

    SuperNova Magnifier & Screen Reader

    SuperNova Magnifier & Screen Reader es un Screen Reader de pago que está disponible en Windows 7, 8.1 y 10. Una de las principales carencias de la aplicación es que sólo está disponible para un navegador, Internet Explorer.

    NVDA

    NVDA es un Screen Reader de donación que está disponible en todas las versiones de Windows a partir de Windows XP (última actualización). La aplicación es compatible con la mayoría de navegadores más conocidos como Google Chrome, Mozilla Firefox, Opera, Internet Explorer, Safari y Microsoft Edge.

    JAWS

    JAWS es un lector de pantalla de pago que cuesta 900 euros y está disponible en Windows 7, 8.1 y 10. Sólo es compatible con Internet Explorer para leer adecuadamente el contenido.

    Read&Write

    Read & Write es un Screen Reader de pago que cuesta 180 £ (200 € aproximadamente) y está disponible en la mayoría de sistemas operativos como Windows, Mac OS, IOS y Android. Además, es compatible con la mayoría de navegadores más utilizados que son Google Chrome, Internet Explorer, Mozilla Firefox, Safari, Opera y Microsoft Edge.
